# Navigator v2 — полный аудит

Дата: 2026-07-10

## Итог

Navigator v2 функционально близок к рабочему продукту: роли, мастер СПН, сделки, карточка, очередь юриста, управление командой, RLS, RPC, Edge Functions, диагностика и CI реализованы.

Главный риск — накопленный технический и операционный долг:

- несколько бизнес-систем используют одну схему `public`;
- рабочий API, диагностика и helpers частично находятся в одной RPC-поверхности;
- Edge deploy ранее выполнялся отдельно от merge;
- карточка сделки и мастер СПН состоят из большого числа patch/guard-модулей;
- role-based сценарии не покрыты E2E;
- все открытые задачи просрочены.

Переписывать проект с нуля не требуется. Порядок улучшений: безопасность и поставка, API/RLS границы, frontend-консолидация, затем рабочие данные.

- `nav-invite-user` задеплоен как version 10, ACTIVE, `verify_jwt=true`;
