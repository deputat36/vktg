## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` сводка документов дополнена индикаторами просрочек:

- `просрочено СПН` — обязательные документы, за которые отвечает СПН;
- `просрочено контроль` — обязательные документы других ответственных специалистов, которые СПН должен контролировать.

Индикаторы показываются только когда соответствующая просрочка есть.
