# Развертывание на GitHub Pages

1. Откройте репозиторий.
2. Перейдите в `Settings` → `Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Save.

После этого GitHub выдаст ссылку вида:

```text
https://deputat36.github.io/vktg/
```

Если сайт не открывается, проверьте:
- есть ли файл `index.html` в корне;
- загружена ли папка `data`;
- нет ли ошибок в консоли браузера.
