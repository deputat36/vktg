# СПН UX: ближайшие действия в карточке сделки

Дата: 26 июня 2026.

## Цель

Сделать работу СПН быстрее и понятнее: показать срочные действия сразу в верхней части карточки сделки, без обязательного перехода во вкладку `Задачи`.

## Что изменено

Добавлен новый модуль:

`assets/js/nav-v2/spn-next-actions-v2.js`

Модуль показывает блок `Ближайшие действия СПН` только для профиля с ролью `spn`.

В блок попадают открытые задачи, которые относятся к СПН:

- `assigned_role = spn`;
- или задача доступна текущему пользователю для изменения статуса.

Задачи сортируются по:

1. сроку выполнения;
2. приоритету;
3. названию.

В блоке показываются:

- срок: просрочено / сегодня / завтра / обычный срок;
- приоритет;
- отметка `можно закрыть СПН` или `контроль`;
- кнопка перехода во вкладку `Задачи`;
- кнопка обновления блока.
