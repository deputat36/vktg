# Статус итерации СПН UX — 2026-06-23 21:40 CEST

## Цель

Снизить риск продолжения работы с устаревшим локальным черновиком сделки.

## Изменения

- Для локального черновика добавлена отдельная метка времени последней работы.
- Свежий черновик показывает, когда его меняли последний раз.
- После 24 часов выводится предупреждение повторно проверить цену, участников, расчеты, расходы и документы.
- Для старых черновиков без временной метки показывается явное предупреждение о неизвестном возрасте.
- Добавлена кнопка «Продолжить», которая переводит СПН непосредственно к мастеру.
- «Начать заново» очищает и данные черновика, и его временную метку.
- При отсутствии полезных данных служебная метка удаляется автоматически.
- Версия `spn-draft-guard-v2.js` в `spn-v2.html` обновлена до `20260623-2145`.
