# SPN UX iteration: role-aware overdue status

Дата: 27 июня 2026.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` уточнена верхняя красная строка просрочек.

Раньше она показывала общий текст про просроченные пункты. Теперь документы разделяются по действию СПН:

- `Документы: СПН N` — документы, которые СПН должен запросить или получить сам;
- `контроль N` — документы других ответственных специалистов;
- для контрольных документов система называет роль, с кем связаться.

Для текущей тестовой карточки это должно давать смысл:

`Документы: СПН 4, контроль 7. Запросите свои документы и свяжитесь с юристом.`

## Подключение

`deal-card-v2.html` подключает модуль как:

`spn-next-actions-v2.js?v=20260627-0245`

Сделать верхнее предупреждение таким же понятным, как детальный блок документов: СПН сразу видит, что делать самому, а где контролировать другого ответственного.
