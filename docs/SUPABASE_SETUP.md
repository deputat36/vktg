# Настройка Supabase

1. Создайте проект Supabase.
2. Откройте SQL Editor.
3. Выполните содержимое `supabase/schema.sql`.
4. Создайте пользователей через Authentication.
5. Заполните таблицу `profiles`.
6. В `config/supabase.js` укажите URL и publishable/anon key.

Важно: не используйте secret key и service_role key в браузерном коде.
