# Статус: аккаунты и ответственные СПН — 2026-06-24 07:55 CEST

## Редактирование аккаунтов

Раздел: `admin-v2.html` → «Команда» → «Пользователи».

Доступ: только роли `owner` и `admin`.

В карточке пользователя редактируются:

- ФИО;
- телефон;
- роль;
- менеджер;
- активность аккаунта.

Кнопка переименована с «Сохранить роль» на «Сохранить данные», поскольку сохраняет все перечисленные поля.

## СПН в списке сделок

Для ролей `owner`, `admin` и `lawyer` в каждой карточке сделки добавлена отдельная строка «Подготовку ведёт».

Отображение:

- один СПН с обеих сторон: `СПН: ФИО`;
- два разных СПН: отдельно СПН продавца и СПН покупателя;
- только одна сторона: СПН соответствующей стороны;
- ответственный не назначен: явная отметка об отсутствии СПН.

Данные берутся из `seller_spn` и `buyer_spn`, уже возвращаемых `nav_v2_get_deals_list`.
