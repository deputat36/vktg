# SPN UX iteration: document action hints

Дата: 27 июня 2026.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` добавлена строка действия СПН внутри каждого документа в верхнем рабочем блоке.

Теперь для каждого документа показывается:

- если ответственный `СПН` — `Действие СПН: запросить, получить или актуализировать этот документ`;
- если ответственный другой специалист — `Действие СПН: проконтролировать ответственного и срок получения`.

## Подключение

`deal-card-v2.html` подключает модуль как:

`spn-next-actions-v2.js?v=20260627-0215`

## Цель

Убрать неопределённость для СПН и новичков: по каждому документу сразу понятно, нужно делать самому или контролировать другого ответственного.

## Область изменений

Изменён только интерфейс Навигатора сделок v2.

CRM «Лидер» не затрагивалась.
