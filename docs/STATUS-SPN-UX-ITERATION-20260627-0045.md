# SPN UX iteration: document ownership hint

Дата: 27 июня 2026.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` добавлена подсказка по ответственности в блоке документов СПН.

Теперь блок явно разделяет:

- `Мои документы СПН` — документы, которые СПН должен запросить, получить или актуализировать сам;
- `Контроль других специалистов` — документы, по которым СПН контролирует срок и при необходимости связывается с ответственным.

Также уточнено пустое состояние блока: если открытых задач и документов нет, СПН может переходить к проверке рисков, актуальности данных или подготовке передачи специалистам.

## Подключение

`deal-card-v2.html` подключает модуль как:

`spn-next-actions-v2.js?v=20260627-0045`

## Цель

Снизить ошибки и ускорить работу СПН за счёт явного разделения личной ответственности и контроля чужих задач.

## Область изменений

Изменён только интерфейс Навигатора сделок v2.

CRM «Лидер» не затрагивалась.
