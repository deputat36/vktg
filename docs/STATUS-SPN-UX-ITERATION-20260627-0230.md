# SPN UX iteration: role-aware priority

Дата: 27 июня 2026.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` уточнён блок `Приоритет сейчас`.

Если есть одновременно:

- просроченные документы СПН;
- просроченные документы других ответственных;

то СПН видит смешанное действие:

`Сначала запросите свои просроченные документы СПН и свяжитесь с юристом по контрольным документам.`

Если просрочены только контрольные документы, система тоже называет роль ответственного.

## Подключение

`deal-card-v2.html` подключает модуль как:

`spn-next-actions-v2.js?v=20260627-0230`

## Цель

Сократить время СПН на определение следующего шага: система сразу говорит, что сделать самому и с кем связаться по контрольным документам.

## Область изменений

Изменён только интерфейс Навигатора сделок v2.

CRM «Лидер» не затрагивалась.
