# СПН UX: группировка документов по ответственности

Дата: 26 июня 2026.

## Цель

Сделать блок `Ближайшие действия СПН` понятнее: СПН должен сразу видеть свои документы отдельно от документов, которые закреплены за другими специалистами, но требуют контроля.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` список обязательных документов разделен на группы:

- `Мои документы СПН`;
- `Контроль других специалистов`.

Для каждой группы показываются первые документы по сроку, а остаток выводится счетчиком.

Сводка по всем документам сохранена:

- всего;
- до задатка;
- до сделки;
- СПН;
- юрист.
