# СПН UX: сводка обязательных документов

Дата: 26 июня 2026.

## Цель

Чтобы СПН не видел только первые два документа и не недооценивал общий объем, в блок `Ближайшие действия СПН` добавлена компактная сводка по обязательным документам.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` добавлена строка сводки перед списком документов:

- всего обязательных документов к контролю;
- сколько нужно до задатка;
- сколько нужно до сделки;
- сколько закреплено за СПН;
- сколько закреплено за юристом.

## Версия

В `deal-card-v2.html` подключена версия:

`spn-next-actions-v2.js?v=20260626-1555`
