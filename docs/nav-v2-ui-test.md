# Навигатор сделок v2 — UI-тест

Цель: проверить путь пользователя без SQL-инструментов.

## Экраны

- spn-v2.html — мастер создания сделки
- deals-v2.html — список сделок
- deal-card-v2.html — карточка сделки
- dashboard-v2.html — рабочий стол
- admin-v2.html — админка

## Тест 1. Простая сделка

В мастере выбрать:

1. Что готовим: Сделка
2. Кого представляем: Только продавца
3. Объект: Квартира
4. Адрес: UI test nav v2
5. Цена: 4500000
6. Задаток: 120000
7. Участники: Один взрослый собственник
8. Документы: ДКП
9. Деньги: Собственные средства
10. Расчеты: СБР, порядок согласован
11. Расходы: согласованы
12. Итог: добавить комментарий СПН

Ожидаемый результат:

- сделка сохраняется;
- открывается карточка;
- карточка помечена как рабочая;
- в списке рабочих сделок она находится поиском;
- рабочий стол пересчитывается.

## Тест 2. Красный сценарий

В мастере добавить:

- ребенок-собственник;
- материнский капитал;
- детские деньги;
- расчеты не согласованы;
- расходы не согласованы.

Ожидаемый результат:

- риск red;
- сделка попадает юристу;
- появляются стоп-факторы и документы по детям.

## Тест 3. Ипотечный сценарий

В мастере добавить:

- ипотека;
- расчеты не согласованы.

Ожидаемый результат:

- риск yellow;
- сделка попадает брокеру;
- появляются ипотечные документы и задачи.

## Что отметить после теста

- ID созданной сделки;
- открылась ли карточка автоматически;
- появились ли документы;
- появились ли задачи;
- изменяются ли статусы;
- добавляются ли комментарии;
- не смешиваются ли демо и рабочие сделки.
