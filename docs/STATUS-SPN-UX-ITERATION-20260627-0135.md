# SPN UX iteration: deal stage document blockers

Дата: 27 июня 2026.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` добавлено предупреждение уровня сделки по обязательным документам:

- `Блокер задатка` — если не хватает документов `до задатка`;
- `Блокер сделки` — если документов до задатка нет, но не хватает документов `до сделки`.

Предупреждение показывает разбиение:

- `СПН` — документы, за которые отвечает СПН;
- `контроль` — документы других ответственных специалистов, которые СПН должен контролировать.

## Подключение

`deal-card-v2.html` подключает модуль как:

`spn-next-actions-v2.js?v=20260627-0135`

## Цель

Снизить риск подготовки к задатку или сделке с неполным пакетом документов и дать СПН быстрый ответ, что именно сейчас блокирует следующий этап.

## Область изменений

Изменён только интерфейс Навигатора сделок v2.

CRM «Лидер» не затрагивалась.
