# СПН UX: ближайшие задачи и обязательные документы

Дата: 26 июня 2026.

## Цель

Сделать карточку сделки полезнее для СПН сразу после открытия: показать не только ближайшие задачи, но и обязательные документы, которые еще не получены или не проверены.

## Что изменено

В `assets/js/nav-v2/spn-next-actions-v2.js` блок `Ближайшие действия СПН` теперь показывает:

- открытые задачи СПН;
- обязательные документы к контролю;
- просроченные сроки по задачам и документам;
- быстрый переход во вкладку `Задачи`;
- быстрый переход во вкладку `Документы`;
- обновление блока.

Документы попадают в блок, если:

- `is_required = true`;
- статус не `received` и не `checked`.

Показываются первые два документа по сроку, остальные считаются и выводятся текстом.
