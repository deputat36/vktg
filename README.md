# Генератор расклеек СПН «Этажи»

Статический инструмент под GitHub Pages для быстрого создания печатных макетов специалистами по недвижимости.

## Главная цель

СПН должен за 1–2 минуты сделать рабочий макет:

1. выбрать задачу;
2. выбрать готовый шаблон;
3. заменить телефон, имя, район, цену и параметры;
4. добавить фото или QR при необходимости;
5. проверить качество;
6. напечатать или сохранить в PDF.

## Что уже есть

- простой интерфейс для СПН;
- расширенные настройки для опытных;
- шаблоны по задачам: куплю, покупатель, объект, новостройки, услуги, аренда, личный бренд, частное;
- плотные, обычные и полупустые шаблоны;
- поддержка 1 фото, 2 фото, планировки, QR;
- деление А4 сверху/снизу, слева/справа и сеткой;
- контроль качества макета;
- сохранение в браузере и экспорт JSON.
