# Быстрое применение патча Навигатора v2

Патч делает ролевое меню и защиту админских страниц.

## Самый простой способ на Windows

1. Скачайте архив.
2. Распакуйте **содержимое архива** в корень локальной папки репозитория `vktg`.
3. В корне папки `vktg` запустите двойным кликом:

```text
APPLY_PATCH_WINDOWS.bat
```

Скрипт сам:

- добавит `role-menu-v2.js`;
- добавит `admin-guard-v2.js`;
- подключит их к HTML-страницам v2;
- выполнит `git add`;
- выполнит `git commit`;
- выполнит `git push origin main`.

## Что проверить после публикации

Под юристом открыть:

```text
https://deputat36.github.io/vktg/dashboard-v2.html
https://deputat36.github.io/vktg/deals-v2.html?filter=lawyer
https://deputat36.github.io/vktg/nav-system-check-v2.html
```

У юриста должно остаться только:

```text
Рабочий стол
Юридическая очередь
Проверка
Выйти
```

Админские страницы под юристом должны показывать «Нет доступа к разделу».
